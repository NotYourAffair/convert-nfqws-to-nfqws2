# Конвертер конфигураций nfqws → nfqws2

## Описание
Утилита для автоматической конвертации конфигурационных файлов и параметров командной строки из nfqws в nfqws2.

## Установка
```bash
git clone https://github.com/NotYourAffair/convert-nfqws-to-nfqws2/nfqws-converter.git
cd nfqws-converter
chmod +x nfqws_converter.py nfqws2-migrate.sh
